# Action Platformer (Demo)

Playable build: https://play.unity.com/api/v1/games/game/5da38c43-c314-4483-b591-c95174bbe60b/build/latest/frame

## Overview
Action Platformer is a Unity 6 demo focused on responsive movement, satisfying combat feedback, and dynamic audiovisual moments.

## Features
- Unity 6 project
- Audio Mixer with separate SFX and Music channels
- New Input System
- Cinemachine camera follow
- Screen shake on player damage and high falls
- Disco ball event: lighting and music shift temporarily when hit by a bullet
- Surface splat decals using masking when enemies die
- Weapons: normal bullets and grenades
- Jetpack (press `J`)
- Safe respawn when the player dies

## Controls (Default)
- Move / Jump / Shoot: bound via the New Input System
- Jetpack: `J`

## Tech Highlights
- Audio Mixer routing for music and SFX separation
- Cinemachine for camera follow and shake
- Mask-based splat effects on enemy death

## Notes
This is a demo project showcasing gameplay systems and visual/audio effects.
